## 🎨 Color Usage Examples

### Gradients:
```tsx
// Main headers and backgrounds
className="bg-gradient-to-br from-[#2a3e35] to-[#1f2e27]"
```

### Solid Colors:
```tsx
// Buttons
className="bg-[#2a3e35] hover:bg-[#1f2e27]"

// Text
className="text-[#2a3e35]"

// Borders
className="border-[#2a3e35]"
```

### With Opacity:
```tsx
// Light backgrounds
className="bg-[#2a3e35]/5"

// Medium backgrounds
className="bg-[#2a3e35]/10"
```
